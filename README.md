# IT Project Manager Toolkit

This repository is my portfolio for IT project delivery and IT service management.

It includes templates, examples, and short notes built while following a free 10-week plan using IBM SkillsBuild, Atlassian University, Cisco NetAcad, Microsoft Learn, and Fortinet training.

## What’s inside

- **templates/** — reusable documents for planning, delivery, reporting, and ITSM basics
- **examples/** — a few filled-in samples to show how I use the templates
- **notes/** — short summaries and cheat sheets from training
- **certificates/** — links or exports for badges/certificates when available

## Why this repo

I’m working toward IT Project Manager / IT Manager roles (Costa Rica; open to remote).  
The goal is straightforward: show practical project habits and solid fundamentals (ops, cloud, security).
