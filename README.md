# IT Management Toolkit

This repository is a portfolio-style collection of practical IT documentation and notes that reflect skills I’ve built through school, work, and structured self-study.

The focus is on **clear communication**, **repeatable processes**, and **realistic artifacts** used in IT projects and IT operations (planning, updates, risk tracking, access changes, incident write‑ups).

## What’s in this repo

- **examples/**  
  Filled artifacts that show how I would document and run real work (projects / IT operations scenarios).

- **notes/**  
  Short topic notes and reference guides from training (kept scannable).

- **templates/**  
  Reusable versions of artifacts.  
  *Rule:* I only add something as a “template” after I’ve used it in an example and I’m confident I can reuse it correctly.

- **certificates/**  
  Links or exports for credentials/badges.
