# IT Project Manager Toolkit

This repo is my working toolkit for IT project delivery and IT service management.

It’s a mix of templates, short examples, and notes built while following a free 10‑week plan using IBM SkillsBuild, Atlassian University, Cisco NetAcad, Microsoft Learn, and Fortinet training.

## What you’ll find here

- **templates/** — reusable documents I’d use to run projects and keep delivery visible
- **examples/** — a few filled-in samples to show how I’d use the templates
- **notes/** — short summaries and cheat sheets from training
- **certificates/** — links or exports for badges/certificates when available

## Why this exists

I’m building toward IT Project Manager / IT Manager roles (Costa Rica; open to remote).  
The goal is simple: show that I can organize work, communicate clearly, and handle the basics of ops, cloud, and security in real projects.

## How to use this repo

- If you’re here to review: start with **templates/**, then check **examples/**.
- If you want to reuse anything: copy a template and adjust it to your environment.
